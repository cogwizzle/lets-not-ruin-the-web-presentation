Replicating Navigation Controls & Navigation

We have to implement our own custom navigation controls that break the way that the web works by default. This is a common pattern in Single Page Applications. Single Page Applications are generally client side rendered.

Examples:
  • Angular/Router
  • React Router or Remix
  • Vaadin
  • Vue Router
Problems:
  • Breaks the native back button
  • Breaks the native forward button
  • Breaks the URL as a form of navigation at times
  • Huge JavaScript payloads to replicate
