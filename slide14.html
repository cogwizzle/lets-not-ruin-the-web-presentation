Return to Simplicity

The solution to the web's complexity issue is to return to simplicity. You don't need a JavaScript framework or library for:

  • Simple Forms handling The web already does this well.
  • Populating data on a page Just render initial data with MVC.
  • Form controls. Why rebuild this?
  • Reusing styles. CSS works just fine. Just make your button look like your button.
  • Simple interactivity. Use Light DOM web components.
  • State Management. Use HATEOAS.

Let the web do what it does really well. Just make it better with JavaScript. Do not reinvent the web.
