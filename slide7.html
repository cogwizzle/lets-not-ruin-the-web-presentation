Breaking the Web

It is not up to the frameworks to decide what technology is used in which way.

Examples:
  • React's incompatibility with Native Web Components.
  • StencilJS's incompatibility with extending elements that are not HTMLElement.

Web standards belong to organizations like the W3C and WHATWG, not to individual companies. It is not up to the frameworks or libraries to decide how the web should be built. This is an overreach and built in obsolescence of a the library or framework.
